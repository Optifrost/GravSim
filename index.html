GravSim – Orbital Physics Sandbox

An interactive gravity simulator with black holes, photons, and 3‑D camera controls.

How to run it locally (step‑by‑step)

  1. Install Python – download and install from python.org (3.7 or newer).
  2. Open a terminal (Command Prompt, PowerShell, or Bash).
  3. Get the code – run: 
    git clone https://github.com/Optifrost/GravSim.git
cd GravSim
  4. Install dependencies – in the terminal: 
    pip install pygame pymunk
  5. Start the program – still in the terminal: 
    python main.py

The window will appear. Try these quick actions:

  • Left‑click empty space → spawn a planet
  • Right‑click empty space → spawn a photon
  • Press B to turn black‑hole mode on/off
  • Scroll mouse wheel or press I/O to zoom
  • Press Space to pause/un‑pause

Controls at a glance

  • Left click – add planet
  • Right click – add photon
  • Click a body – select it
  • Delete – delete selected body
  • Backspace – delete last placed planet
  • Space – pause/resume
  • Mouse wheel / I/O – zoom
  • Arrow keys – rotate camera
  • WASD – pan camera
  • +/- – increase/decrease gravity
  • M/N – increase/decrease selected planet mass
  • R/F – increase/decrease selected planet size
  • C – clear scene
  • Home – reset camera view
  • B – toggle black‑hole mode
  • H – hide/show UI

Enjoy experimenting with orbits!
