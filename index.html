🌌 StarDance Orbital Physics Simulator

A desktop-based gravitational simulator featuring realistic stellar textures, black hole physics, and interactive orbital mechanics.

✨ Key Features

  • 🌞 Realistic sun texture (replaces plain yellow circle)
  • ⚫ Black hole mode with accretion disk (press 'B')
  • 🎯 Stable physics - objects consumed by black holes rather than flying off
  • ⚡ Velocity limits to prevent numerical explosions
  • 🎮 Full 3D camera controls and interactive parameters

🚀 How to Run Locally

This is a desktop application - follow these steps to run it on your computer:

git clone https://github.com/Optifrost/GravSim.git cd GravSim pip install pygame pymunk python main.py

Requires: Python 3.7+, Pygame, Pymunk

🎮 Controls

Left Click: Add planet
Right Click: Add photon
Click Body: Select for modification
Delete: Remove selected body
Backspace: Remove latest planet
Space: Pause/Resume
Mouse Wheel / I/O: Zoom
Arrow Keys: Rotate camera
WASD: Pan camera
+ / -: Adjust gravity
M / N: Adjust selected planet mass
R / F: Adjust selected planet size
C: Clear scene
Home: Reset camera view
B: Toggle black hole mode
H: Hide/show UI

🔭 What Makes This Special

Unlike basic gravity simulators, StarDance includes:

  • Realistic stellar visualization with texture mapping
  • Physically-inspired black hole consumption (not just visual effects)
  • Photon particles that behave like light (massless, speed-limited)
  • Interactive parameter adjustment in real-time
  • 3D camera navigation for viewing orbits from any angle
